## Direct Command Shortcuts

```bash
# Skip interactive, use preset theme
omgkit init my-project --theme neo-tokyo

# From screenshot (non-interactive)
omgkit init my-project --theme-from ~/design.png

# From URL (non-interactive)
omgkit init my-project --theme-from https://linear.app

# Default minimal theme
omgkit init my-project --theme minimal-slate

# No theme (skip design system)
omgkit init my-project --no-design
```

---

## Theme Management Commands (Post-Init)

```bash
# List available themes
/design:themes

# Switch to different theme
/design:theme electric-cyan

# Preview current theme
/design:preview

# Add components
/design:add button card dialog

# Extract from new screenshot
/design:from-screenshot ~/new-inspiration.png

# Reset to original theme
/design:reset
```
