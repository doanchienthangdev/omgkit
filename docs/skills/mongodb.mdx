## Query Patterns
```javascript
// Find
db.users.find({ email: /example.com/ });

// Aggregation
db.users.aggregate([
  { $match: { active: true } },
  { $group: { _id: "$role", count: { $sum: 1 } } }
]);

// Update
db.users.updateOne(
  { _id: id },
  { $set: { name: "New Name" } }
);
```
