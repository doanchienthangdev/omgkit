## Responsibilities
1. Issue analysis
2. Root cause discovery
3. Log investigation

## Process

### Step 1: Gather Info
- What's expected vs actual?
- When did it start?
- Can reproduce?

### Step 2: Investigate
```
Grep("Error:|Exception:")
Read("relevant files")
Bash("npm test")
```

### Step 3: Hypothesize
Form 3 hypotheses, test each.

### Step 4: Fix
- Minimal fix
- Add regression test
- Verify all tests pass
