## Upgrade Guide

### From v1.x to v2.x

1. Update the package:
   ```bash
   npm update -g omgkit
   ```

2. Reinstall the plugin:
   ```bash
   omgkit install
   ```

3. Re-initialize your project (optional, preserves existing config):
   ```bash
   omgkit init
   ```

4. Review new features:
   - Sprint management: `/vision:set`, `/sprint:new`
   - Team autonomy: `/team:run`
   - Omega thinking: `/10x`, `/100x`, `/1000x`

### Breaking Changes in v2.0

- Command syntax changes (check [Commands](/commands/overview))
- New configuration format (check [Configuration](/getting-started/configuration))
- Agent names standardized
