# Marketing Content Workflow

## Workflow Steps

### Step 1: Research
**Agent:** @researcher
**Command:** `/planning:research`

- Target audience analysis
- Competitor research
- Market positioning
- Key messages

### Step 2: Ideation
**Agent:** @brainstormer
**Command:** `/planning:brainstorm`

- Message angles
- Headline options
- Value propositions
- Call-to-actions

### Step 3: Content Creation
**Agent:** @copywriter

- Write headlines
- Create body copy
- Features/benefits
- CTAs

### Step 4: Review
**Agent:** @copywriter

- Tone consistency
- Clarity
- Persuasiveness
- Grammar
