## Quick Start

```bash

```


# üìù Commit: [message]

Create conventional commit.

## Process
1. Stage changes
2. Analyze diff
3. Generate commit message
4. Commit

## Commit Format
```
<type>(<scope>): <subject>

<body>
```

## Types
- feat: New feature
- fix: Bug fix
- docs: Documentation
- style: Formatting
- refactor: Code restructure
- test: Tests
- chore: Maintenance
