## Quick Start

```bash

```

Create implementation plan for: **\<task\>**

## Process
1. Understand requirements
2. Research codebase
3. Design solution
4. Break into tasks

## Output
Save plan to `plans/<feature>.md`:

```markdown
# Plan: [Feature]

## Overview
[Brief description]

## Tasks
1. [ ] [Task] - [2-5 min] - [location]
2. [ ] [Task] - [2-5 min] - [location]

## Files
- `path/file.ts` - [changes]

## Testing
- Unit: [what to test]
- Integration: [flows]

## Risks
| Risk | Mitigation |
```
