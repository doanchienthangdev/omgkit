## Output
Save plan to `plans/\<feature>.md`:

```markdown
# Plan: [Feature]

## Overview
[Brief description]

## Tasks
1. [ ] [Task] - [2-5 min] - [location]
2. [ ] [Task] - [2-5 min] - [location]

## Files
- `path/file.ts` - [changes]

## Testing
- Unit: [what to test]
- Integration: [flows]

## Risks
| Risk | Mitigation |
```
