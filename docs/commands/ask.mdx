## Overview

Quick question with codebase context

## Quick Start

```bash
/ask "question"
```


# ‚ùì Ask: $ARGUMENTS

Answer: **$ARGUMENTS**

## Process
1. Understand the question
2. Search codebase for context
3. Analyze relevant code
4. Provide direct answer

## Response Format
- Direct answer first
- Supporting evidence
- Code references if relevant
- Recommendations if applicable



## Tools Used

This command uses the following tools:

- **Read** - Enables read capabilities
- **Grep** - Enables grep capabilities
- **Glob** - Enables glob capabilities


## Examples

### Basic Usage

```bash
/ask "your input here"
```

### With Context

```bash
# First, ensure context is loaded
/index

# Then run the command
/ask "detailed description"
```

## Tips

<Note>
For best results, be specific in your descriptions and include relevant file paths or context.
</Note>
