## Overview

Display comprehensive ML project status including data, model, deployment, and pipeline health

## Quick Start

```bash
/omgml:status "[--detailed]"
```


# ML Project Status: "[--detailed]"

Show ML project status: **"[--detailed]"**

## Agent
Uses **orchestrator-agent** for comprehensive status collection.

## Parameters
- **--detailed**: Show extended information for each component

## Status Components

### Data Status
- Training set size and last update
- Validation/Test set sizes
- Data version (DVC)
- Data quality metrics

### Model Status
- Current model version
- Model architecture
- Performance metrics (accuracy, F1, etc.)
- Last training timestamp

### Deployment Status
- Production model version
- Staging model version
- Endpoint health
- Request latency

### Pipeline Status
- Last pipeline run
- Success/failure rate
- Next scheduled run
- Active experiments
