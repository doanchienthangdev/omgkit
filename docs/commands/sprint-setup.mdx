# Sprint Setup Workflow

Setup sprint: **\<sprint name\>**

## Workflow Steps

### Step 1: Vision Review
**Agent:** @sprint-master
**Command:** `/vision:show`

- Review product vision
- Check current progress
- Identify priorities
- Align with stakeholders

### Step 2: Backlog Grooming
**Agent:** @planner
**Command:** `/backlog:show`

- Review backlog items
- Estimate effort
- Prioritize by value
- Identify dependencies

### Step 3: Sprint Planning
**Agent:** @sprint-master
**Command:** `/sprint:new "\<sprint name\>"`

- Select sprint items
- Define sprint goal
- Assign to agents
- Set capacity

### Step 4: Task Breakdown
**Agent:** @planner

- Break down stories
- Create implementation tasks
- Define acceptance criteria
- Identify risks

### Step 5: Kickoff
**Agent:** @sprint-master
**Command:** `/sprint:start`

- Communicate sprint goal
- Confirm assignments
- Set up tracking
- Start sprint

## Progress Tracking
- [ ] Vision reviewed
- [ ] Backlog groomed
- [ ] Sprint planned
- [ ] Tasks broken down
- [ ] Sprint started

Execute each step. Sprint ready to begin.
