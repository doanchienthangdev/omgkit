## Overview

Interactive brainstorming session

## Quick Start

```bash
/brainstorm
```

## Process
1. Define problem clearly
2. Generate 10+ ideas (no judgment)
3. Group and combine
4. Evaluate top options

## Methods
- Divergent thinking
- Lateral thinking
- SCAMPER technique
- Reverse brainstorming

## Output
```markdown

1. [Idea] - [rationale]
...
```
