## Overview

Interactive brainstorming session

## Quick Start

```bash
/planning:brainstorm "topic"
```


# ðŸ’¡ Brainstorm: \<topic\>

Creative exploration for: **\<topic\>**

## Process
1. Define problem clearly
2. Generate 10+ ideas (no judgment)
3. Group and combine
4. Evaluate top options

## Methods
- Divergent thinking
- Lateral thinking
- SCAMPER technique
- Reverse brainstorming

## Output
```markdown
## Ideas
1. [Idea] - [rationale]
...

## Top 3 Recommendations
1. **Best Overall**: [why]
2. **Most Innovative**: [why]
3. **Quickest Win**: [why]
```
