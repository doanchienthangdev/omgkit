## Usage Patterns

### Basic Usage

```bash
/command argument
/command "argument with spaces"
/command --flag value
```

### Common Workflows

```bash
# Feature development
/feature "add user authentication"

# Quick bug fix
/fix "login not working"

# Full planning
/plan "checkout flow" && /execute-plan

# Git workflow
/commit && /pr
```
