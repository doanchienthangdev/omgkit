## Context

Context commands manage session state, project indexing, and parallel task execution.

| Command | Description | Agents Used |
|---------|-------------|-------------|
| [`/context:checkpoint`](/commands/checkpoint) | Save or restore session state for later resumption | - |
| [`/context:index`](/commands/index) | Generate comprehensive project index for codebase awareness | scout |
| [`/context:load`](/commands/load) | Load project context from OMEGA.md and config files | - |
| [`/context:mode`](/commands/mode) | Switch behavioral mode (omega, tutor, implementation, etc.) | - |
| [`/context:spawn`](/commands/spawn) | Launch parallel background task with specified agent | Any agent |
| [`/context:spawn-collect`](/commands/spawn-collect) | Collect and aggregate results from parallel spawned tasks | - |

<Tip>
  Use `/context:index` at the start of a session for better codebase awareness.
</Tip>

## Usage Patterns

### Basic Usage

```bash
# Simple command
/dev:fix "login button not working"

# Command with detailed description
/dev:feature "add user authentication with OAuth2 support"

# Command with flags
/sprint:team-run --mode full-auto
```

### Common Workflows

<AccordionGroup>
  <Accordion title="Feature Development">
    ```bash
    # Option 1: Full automated workflow
    /dev:feature "add shopping cart functionality"

    # Option 2: Step-by-step
    /planning:plan "shopping cart"
    /planning:execute-plan
    /dev:test
    /dev:review
    /git:ship
    ```
  </Accordion>

  <Accordion title="Bug Fixing">
    ```bash
    # Quick fix
    /dev:fix-fast "typo in header"

    # Complex bug
    /dev:fix-hard "race condition in checkout"

    # From logs
    /dev:fix-logs
    ```
  </Accordion>

  <Accordion title="Sprint Management">
    ```bash
    # Initialize project
    /sprint:init

    # Set vision and create sprint
    /sprint:vision-set
    /sprint:sprint-new --propose

    # Run AI team
    /sprint:team-run --mode semi-auto
    ```
  </Accordion>

  <Accordion title="Omega Thinking">
    ```bash
    # Find improvements
    /omega:10x "reduce API response time"
    /omega:100x "user onboarding experience"

    # Apply principles
    /omega:principles
    ```
  </Accordion>
</AccordionGroup>
