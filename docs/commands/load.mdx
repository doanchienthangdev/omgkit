## Overview

## Quick Start

```bash
/context:load "component"
```

## Components
- `all` - Full project context
- `config` - Configuration files
- `api` - API routes
- `components` - UI components
- `tests` - Test files



## Tools Used

This command uses the following tools:

- **Read** - Enables read capabilities
