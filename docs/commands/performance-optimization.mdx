# Performance Optimization Workflow

Optimize: **\<optimization target\>**

## Workflow Steps

### Step 1: Profiling
**Agent:** @fullstack-developer

- Profile application
- Identify bottlenecks
- Measure baselines
- Document hotspots

### Step 2: Analysis
**Agent:** @architect

- Analyze root causes
- Evaluate solutions
- Assess trade-offs
- Prioritize fixes

### Step 3: Optimization
**Agent:** @fullstack-developer
**Command:** `/quality:optimize`

- Implement optimizations
- Add caching
- Optimize queries
- Reduce bundle size

### Step 4: Validation
**Agent:** @tester

- Benchmark improvements
- Load testing
- Compare to baseline
- Verify no regressions

### Step 5: Documentation
**Agent:** @docs-manager

- Document changes
- Update runbooks
- Share learnings

## Progress Tracking
- [ ] Profiling complete
- [ ] Analysis done
- [ ] Optimizations applied
- [ ] Improvements validated
- [ ] Documented

Execute each step. Measure everything.
