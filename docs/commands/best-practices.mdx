# Best Practices Workflow

Research best practices for: **\<topic for best practices\>**

## Workflow Steps

### Step 1: Current State
**Agent:** @researcher

- Analyze current practices
- Identify gaps
- Document issues
- Set improvement goals

### Step 2: Research
**Agent:** @researcher
**Command:** `/planning:research "best practices \<topic for best practices\>"`

- Industry standards
- Expert recommendations
- Case studies
- Common patterns

### Step 3: Synthesis
**Agent:** @oracle

- Synthesize findings
- Adapt to context
- Prioritize practices
- Create guidelines

### Step 4: Documentation
**Agent:** @docs-manager

- Create style guide
- Document patterns
- Add examples
- Create checklists

### Step 5: Implementation
**Agent:** @planner

- Plan adoption
- Create training
- Set up enforcement
- Track compliance
