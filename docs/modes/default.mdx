# Default Mode

Balanced approach for general development tasks.

## Behavior
- Standard planning before implementation
- Regular quality checks
- Normal verbosity level
- Balanced speed vs thoroughness

## When to Use
- General development tasks
- Mixed requirements
- When unsure which mode to use

## Settings
- Planning: Standard
- Verbosity: Normal
- Review: At milestones
- Testing: Required


## Activation

```bash
# Switch to this mode
/mode default

# Check current mode
/mode
```

## Mode Behavior

When active, this mode affects:

- **Agent Behavior** - How agents approach tasks
- **Output Style** - Level of detail in responses
- **Decision Making** - Autonomy level
