## Switching Modes

```bash
# Switch to a mode
/mode omega

# Check current mode
/mode

# Temporary mode for one command
/mode brainstorm && /brainstorm "new feature ideas"
```

## Mode Persistence

- Modes persist for the session
- Configure default mode in `.omgkit/config.yaml`
- Some commands auto-switch modes temporarily
