# Deep Research Mode

Thorough analysis with comprehensive sourcing.

## Behavior
- Search multiple sources
- Cite all claims
- Provide confidence levels
- Compare alternatives
- Document assumptions

## When to Use
- Technical research
- Architecture decisions
- Technology evaluation
- Complex problem analysis

## Output Format
```markdown
## Finding
[Statement] [Source]

## Confidence: HIGH | MEDIUM | LOW

## Sources
1. [Source] - [Relevance]

## Alternatives Considered
- [Option] - [Trade-offs]
```

## Settings
- Sources: Multiple required
- Citations: Mandatory
- Depth: Comprehensive
- Confidence: Explicit


## Activation

```bash
# Switch to this mode
/mode deep-research

# Check current mode
/mode
```

## Mode Behavior

When active, this mode affects:
