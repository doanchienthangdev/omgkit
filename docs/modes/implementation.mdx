# Implementation Mode

Optimized for writing code efficiently.

## Behavior
- Code-first responses
- Minimal explanations
- Execute plans directly
- Focus on output
- Fast iteration

## When to Use
- Executing plans
- Writing code
- Making changes
- Bug fixes

## Output
```typescript

  // Only essential comments
}
```

## Settings
- Output: Code primary
- Comments: Essential only
- Explanations: Minimal
- Speed: Prioritized
